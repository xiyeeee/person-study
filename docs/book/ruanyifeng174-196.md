2. [使用 React.memo() 提高组件性能](https://alexsidorenko.com/blog/react-render-always-rerenders/)

   默认情况下，如果 React 父组件重新渲染，它包含的所有子组件都会重新渲染，即使子组件没有任何变化。

   React.memo() 方法可以防止子组件不必要的渲染，从而提高组件性能。
